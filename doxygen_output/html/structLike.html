Detailed Description

how much a species is liked by another

the liked_spec_id is the species id this species like and the like factor is how much he likes it. there is no way to know to which species belogns this like because they were inside the speciesinfo likings so it is enstablished that this like belongs to a precise species. in the example rabbit example: if this liking is referred to a rabbit the liked_spec_id is the id of a carrot and the like_factor is how much from 1 to 100 the rabbit likes the carrot
