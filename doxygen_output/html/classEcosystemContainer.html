void Eco::m_bound_translator ( int &  x,
int &  y 
) [private]

translate the coordinate if there were no boundaries

if there were no boundaries the x and y coord could be more than m_x_size or m_y_size or less than 0. this function provide a translation of such numbers in an interval from 0 - m_x/y_size.

if the m_boundaries flag is not true and the coordinates are not in the interval it will print an error message and set both x and y to 0 example: if x = m_x_size+1 it becomes 0
